Usage:
java -jar lib/TextProcess.jar [options] <inputfile>
 -A   print all below
 -b   print the bigram count of the input file
 -f   print the word count of the input file
 -p   print the palindrome of the input file

Example:
we have:
do geese see god abba bat tab
a a c a a
java -jar lib/TextProcess.jar -A README.txt
